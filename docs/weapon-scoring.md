# 전설 무기 점수표

## 사용 안내

- 전설 무기 점수 계산은 **모든 옵션 수치 MAX = 100점** 기준입니다.
- 점수는 **'최대 옵션에 얼마나 가까운가'** 를 나타냅니다.
- 점수는 **강화 수치와 무관**합니다.
- 스킬형 무기는 점수가 높더라도 실제 딜량 1순위를 의미하지는 않습니다.
  - 실제 딜량은 스킬 구조, 히트 수, 전투 환경에 따라 달라질 수 있습니다. (특히 화이트스노우)
- 스킬/확률 요소가 없는 무기는 점수와 실제 데미지 효율이 거의 동일하게 반영됩니다.
- 공격속도가 이미 충분해 필요가 없는 유저는 점수가 낮더라도 그 자체로 졸업일 수 있습니다.

---

## 등급 기준표

| 등급 | 점수 범위 |
|------|-----------|
| 졸업 | 95.0 ~ 100 |
| 최상 | 92.0 ~ 94.9 |
| 상   | 88.0 ~ 91.9 |
| 중상 | 84.0 ~ 87.9 |
| 중   | 78.0 ~ 83.9 |
| 중하 | 70.0 ~ 77.9 |
| 하   | 0 ~ 69.9 |

> 본 등급은 개인적인 참고 기준입니다.
> 최상·상급 옵션도 충분히 우수하지만, 졸업 타이틀의 상징성과 희소성을 고려하여 95점 이상을 졸업 기준으로 설정했습니다.

---

## 1. 신의손

### 계산식

```
점수 = ((1 + 무기데미지%/100) × (1 + 빛기술피해%/100) × 노바레벨계수) ÷ 41.888 × 100
```

### MAX 옵션

| 옵션 | 최대값 |
|------|--------|
| 무기데미지 | 300% |
| 빛기술피해 | 40% |
| 신성한노바 | 8 Lv |

### 신성한 노바 레벨 계수

| 레벨 | 계수 |
|------|------|
| 4 Lv | 6.92 |
| 5 Lv | 7.06 |
| 6 Lv | 7.20 |
| 7 Lv | 7.34 |
| 8 Lv | 7.48 |

### 계산 예시

> 무기데미지 278%, 빛기술피해 36.9%, 노바 5레벨(계수 7.06)
>
> ```
> ((1 + 278/100) × (1 + 36.9/100) × 7.06) ÷ 41.888 × 100
> = 87.22점 → 중상급
> ```

## 2. 세계의 파괴자

### 계산식

```
점수 = ((1 + 무기데미지%/100) × (1 + 치명타피해%/100) × (1 + 물리기술피해%/100)) ÷ 8.976 × 100
```

### MAX 옵션

| 옵션 | 최대값 |
|------|--------|
| 무기데미지 | 340% |
| 치명타피해 | 70% |
| 물리기술피해 | 20% |

---

## 3. 화이트 스노우

### 계산식

```
점수 = ((1 + 무기데미지%/100) × (1 + 공격속도%/100) × 눈보라계수 × 얼음구슬계수) ÷ 268.164 × 100
```

### MAX 옵션

| 옵션 | 최대값 |
|------|--------|
| 무기데미지 | 300% |
| 공격속도 | 30% |
| 눈보라 | 8 Lv |
| 얼음구슬 | 8 Lv |

### 눈보라 레벨 계수

| 레벨 | 계수 |
|------|------|
| 4 Lv | 16.0 |
| 5 Lv | 16.5 |
| 6 Lv | 17.0 |
| 7 Lv | 17.5 |
| 8 Lv | 18.0 |

### 얼음구슬 레벨 계수

| 레벨 | 계수 |
|------|------|
| 4 Lv | 2.485 |
| 5 Lv | 2.580 |
| 6 Lv | 2.675 |
| 7 Lv | 2.770 |
| 8 Lv | 2.865 |

## 4. 사자

### 계산식

```
점수 = ((1 + 무기데미지%/100) × (1 + 공격속도%/100) × 번개노바계수 × 시전확률) ÷ 13.104 × 100
```

### MAX 옵션

| 옵션 | 최대값 |
|------|--------|
| 무기데미지 | 320% |
| 공격속도 | 30% |
| 번개노바 | 5 Lv |
| 시전확률 | 60% |

### 번개노바 레벨 계수

| 레벨 | 계수 |
|------|------|
| 2 Lv | 3.625 |
| 3 Lv | 3.750 |
| 4 Lv | 3.875 |
| 5 Lv | 4.000 |

### 시전확률 입력값

| 확률 | 입력값 |
|------|--------|
| 60% | 0.60 |
| 45% | 0.45 |

---

> 주로 사용되는 4종 무기 계산식입니다.
> 최대치가 정확히 확인되면 다른 무기들도 추가 예정입니다.
